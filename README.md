# Workhorse AI — Workplace Productivity Assistant

Workhorse AI is an AI-powered productivity assistant that helps professionals save time and produce higher-quality work. It combines a free-form chatbot with focused, task-specific tools for the most common workplace jobs.

> Built with React + Vite + Tailwind on the front end, and a streaming Edge Function powered by the Lovable AI Gateway on the back end.

---

## ✨ Features

- 💬 **AI Chatbot** — free-form conversation with streaming responses
- ✉️ **Write Email** — draft polished emails with adjustable tone (Formal / Informal / Persuasive)
- 📝 **Summarize Notes** — turn meeting notes into key points, decisions, action items & deadlines
- ✅ **Plan My Day** — prioritize tasks (High/Medium/Low) into a structured schedule
- 🔎 **Research Assistant** — get an in-depth briefing on any topic
- ⚡ **Streaming responses** via Server-Sent Events (SSE)
- 🎨 Clean, responsive UI with a semantic design system (light/dark ready)

---

## 🧠 How It Works

The app sends user input to a Supabase Edge Function (`supabase/functions/chat`), which forwards the request to the **Lovable AI Gateway** (`google/gemini-3-flash-preview` by default) with a system prompt that enforces the assistant's tone, structure, and responsible-AI guidelines. Responses stream back to the UI token-by-token.

## 🛠 Tech Stack

- **Frontend:** React 18, Vite 5, TypeScript, Tailwind CSS, shadcn/ui, lucide-react, react-markdown
- **Backend:** Supabase Edge Functions (Deno)
- **AI:** Lovable AI Gateway (Gemini / GPT models)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ or Bun
- A Supabase project (or Lovable Cloud, which provides one automatically)
- A `LOVABLE_API_KEY` configured as a secret in your Supabase project

### Install & run

```bash
# Install dependencies
bun install   # or: npm install

# Start the dev server
bun run dev   # or: npm run dev
```

Open http://localhost:5173 in your browser.

### Environment Variables

A `.env` file is auto-generated by Lovable Cloud and contains:

```
VITE_SUPABASE_URL=...
VITE_SUPABASE_PUBLISHABLE_KEY=...
VITE_SUPABASE_PROJECT_ID=...
```

The Edge Function requires this secret in Supabase:

```
LOVABLE_API_KEY=...
```

## ⚠️ Responsible AI

Workhorse follows responsible-AI guidelines:
- Does not fabricate facts and indicates uncertainty when relevant
- Avoids biased or inappropriate language
- Encourages user validation for critical decisions

> Workhorse can make mistakes. Always verify important information before acting on it.

## 📦 Deployment

This project is built and deployed through [Lovable](https://lovable.dev). You can also self-host:

1. Connect the project to GitHub from Lovable
2. Clone the repo and deploy the frontend to any static host (Vercel, Netlify, Cloudflare Pages, etc.)
3. Keep the Supabase Edge Function deployed (or migrate it to your own backend)
